# Get Better – Personal Productivity Dashboard

Get Better is a personal productivity and self-improvement web application built with HTML, CSS, and JavaScript. It helps users organize daily tasks, build habits, track goals, monitor progress, and improve productivity through an intuitive and modern interface.

## Features

- Personal productivity dashboard
- Daily task management
- Habit tracker
- Weekly planning
- Goal tracking with progress indicators
- Monthly goals management
- Statistics and progress charts
- Built-in clock and date display
- Responsive and modern UI
- Local data storage
- Dark-themed interface

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Chart.js

## Project Highlights

- Clean and responsive dashboard design
- Interactive habit tracking
- Goal progress visualization
- Daily and weekly planning tools
- Dynamic charts and statistics
- User-friendly interface
- Local browser storage for saving data

## How to Run

1. Clone or download the repository.
2. Open `Get_better.html` in any modern web browser.
3. No installation or external server is required.
