# 🤖 RoboMatrix Team Website

یک وب‌سایت مدرن و تعاملی برای معرفی اعضای تیم RoboMatrix که با React طراحی شده است. این پروژه دارای رابط کاربری مدرن، انیمیشن‌های روان و طراحی ریسپانسیو بوده و اطلاعات اعضای تیم، معرفی پروژه و نمونه کد را نمایش می‌دهد.

## ✨ امکانات

- 👥 نمایش اعضای تیم به صورت کارت‌های تعاملی
- 📄 نمایش رزومه هر عضو در مودال اختصاصی
- 🎨 طراحی مدرن با افکت Glassmorphism
- 🌌 پس‌زمینه متحرک (Particle Animation)
- 📱 طراحی کاملاً Responsive
- 🍔 منوی کشویی (Drawer Navigation)
- 💻 نمایش کد با Syntax Highlighting (Prism.js)
- 📋 امکان کپی کردن کد
- ⚡ انیمیشن‌های روان برای صفحات و المان‌ها

---

## 🛠️ تکنولوژی‌های استفاده شده

- React 18
- JavaScript (ES6+)
- HTML5
- CSS3
- Prism.js
- Babel Standalone

## 🚀 اجرای پروژه

کافی است پروژه را Clone کنید:

```bash
git clone https://github.com/USERNAME/REPOSITORY.git
```

سپس وارد پوشه پروژه شوید:

```bash
cd REPOSITORY
```

و فایل `index.html` را در مرورگر اجرا کنید یا از افزونه **Live Server** در VS Code استفاده کنید.
